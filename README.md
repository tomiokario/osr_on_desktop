# osr_on_desktop
Desktopから画像のテキストをOSRで読み取り，クリップボードにコピーするスクリプト

事前準備
- 必要ライブラリをインストール
- osrとosr.pyをデスクトップに配置する

使い方
- osr（Unix実行ファイル）をダブルクリック
- 言語を選択
- osrしたい画像をドラッグ&ドロップしてpathを入力
